创建过程

blog是基于hexo生成的静态网站
原本是要完全自己实现，但做得实在太丑，只能站在巨人的肩膀上仰望星空
故以flash fox的hexo主题为模板进行适当修改
暂时部署在github及vercel上，待后期域名备案成功再转移到服务器上

1.安装nodejs及git

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)

  • Git

    因为国内nodejs挺慢的，建议下载 淘宝 Node.js 镜像

2.安装hexo

在git bash中

$ cd /d      //D盘根目录下安装
$ npm install -g hexo-cli
3.使用

_config.yml
网站的 配置 信息，您可以在此配置大部分的参数，例如主题及网站名。

source
资源文件夹是存放用户资源的地方。主要在此处寻找和修改页面信息。

themes
主题文件夹。Hexo 会根据主题来生成静态页面。

常用命令
在git bash中

$ cd /d/hexo     //进入hexo目录
$ hexo new [layout] <title>  // 新建文章
$ hexo generate //生成静态文件 可简写为hexo g
$ hexo deploy  //部署网站 可简写为hexo d
$ hexo clean  //清除缓存文件 (db.json) 和已生成的静态文件 (public)
4.部署

安装 hexo-deployer-git。

$ npm install hexo-deployer-git --save

在 _config.yml 中修改参数

deploy:
type: git
repo: #https://bitbucket.org/JohnSmith/johnsmith.bitbucket.io
branch: [branch]
message: [message]

Hexo 提供了快速方便的一键部署功能，只需以下命令就能将网站部署到服务器上。

$ hexo clean
$ hexo g
$ hexo d

以上即可部署完成

由于github速度实在过慢 挂在vercel上效果会更好
